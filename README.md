#**Job Tracker**

A focused job tracking app to manage your applications and keep your pipeline organized.

Track where you’ve applied, what’s moving forward, and what’s dead — all in one place.

---

##**Overview**

This app lets you:

Add and manage job applications
Track progress across different hiring stages
View everything in a centralized dashboard
Update statuses as your application moves forward

##**Tech Stack**

###Frontend

React
HTML
CSS

###Backend

Node.js
Express

###Database

PostgreSQL

##**Why this exists**

Job hunting gets messy fast.

Spreadsheets break. Notes get lost. You forget where you applied.

This app keeps everything structured so you can focus on actually getting hired.
